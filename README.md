# info
this repo is advanced and fixed version of script named "undiscord". yeah, that famous one.

# usage
you can just copy the code and execute at chrome dev console on discord web app. trash icon will appear at dm's header. click the trash icon, fill the blanks by clicking "me" and "current" buttons.
afterwards click "start delete" and it will start to delete your messages.
if there's any errors don't hesitate to create an issue.
